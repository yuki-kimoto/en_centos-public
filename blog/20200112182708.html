ApacheでHTTPS接続のためにmod_sslをインストールする - CentOS 7

CentOS 7のApacheでHTTPS接続のためにmod_sslをインストールしましょう。

HTTPS接続は、SSL接続ですので、mod_sslが必要です。

yumコマンドを使って、mod_sslをインストールしましょう。 

  sudo yum -y install mod_ssl
