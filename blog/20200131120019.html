Apache error log location - CentOS 7

The apache error log is located at: 

/var/log/httpd/error_log

By default, the error logs are automatically routed, and the four past logs are also in the same directory. 

error_log-20200112
error_log-20200119
error_log-20200126
error_log-20200202

How to check the error log

As a method to check the error log, I will introduce a Linux command that can output and check the contents of the file.

  • head command --output the beginning of the file
  • tail command-output the end of the file
  • cat --output the contents of the file
  • vi --Text editor for command line
